Git

  • ls (List files in the directory) — выводит список файлов и каталогов в директории.
  • cd (Change directory) — меняет текущую директорию.
  • pwd (Present working directory) — показывает текущий рабочий каталог.
  • mkdir (Make directory) — создаёт новую директорию.
  • mv (Move) — перемещает или переименовывает файлы и каталоги.
